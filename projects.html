Student Portal

This Java Swing application enables students to view attendance records, grades, and profile details. Instructors can enter and update attendance, grades per module, and admin notes on profiles. Data is stored securely in a CSV database. Key features include student login, admin settings, an intuitive UI, and seamless navigation. Overall, the system delivers robust digital tools for managing student data and monitoring academic performance, demonstrating strong Java development skills.

Snake Game

This Project was made using Java Swing and AWT. This snake game is a classic game that is played with the keyboard. The player controls the snake by pressing the arrow keys. The goal of the game is to eat as many apples as possible without hitting the walls or its own tail. The snake grows in length as it eats apples, making it more difficult to control. The game ends when the snake hits a wall.

Tic Tac Toe

A classic tic-tac-toe game made with Java. Players take turns placing their marks (X or O) on a 3x3 grid. The first player to get three of their marks in a row, column, or diagonal wins. The game is simple to learn but difficult to master, and it is sure to provide hours of entertainment for players of all ages
